## 🚨 Team Git Rules

- ❌ Never push directly to `main`
- ✅ Always create a new branch for your feature (`feature/<your-task-name>`)
- 📤 Push your code to your own branch only
- 🛠️ Open a Pull Request (PR) for review
- ⏳ Wait for approval before merging
